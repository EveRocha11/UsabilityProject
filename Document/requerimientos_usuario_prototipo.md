
# Requerimientos

A continuación se presentan 9 historias de usuario (HU), cada una con su tabla individual, tareas y criterios de aceptación.



| Identificador | HU1 |
|---------------|-----|
| Estimación (días) | 3 |
| Prioridad | Alta |
| Título | Instrucciones Claras |
| Descripción | Como usuario, quiero que todas las actividades tengan instrucciones claras y visibles, para poder entender fácilmente qué debo hacer. |
| Criterios de aceptación | - Las instrucciones son visibles y comprensibles en cada pantalla.<br>- Se incluyen ejemplos o tutoriales breves donde sea necesario.<br>- Los usuarios reportan comprensión sin ayuda externa. |
| Tareas | - Redactar instrucciones claras para cada actividad.<br>- Incluir ejemplos visuales o tutoriales.<br>- Validar comprensión con usuarios. |
